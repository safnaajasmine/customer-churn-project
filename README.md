# Customer Churn Prediction

## 📌 Project Overview
This project predicts whether a customer will churn using Machine Learning.

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## 📊 Machine Learning Models
- Logistic Regression
- Decision Tree
- Random Forest
- Gradient Boosting
- XGBoost
- KNN
- Naive Bayes

## 🔍 Process
- Data Cleaning (Converted TotalCharges to numeric)
- No missing values after preprocessing
- Label Encoding
- Train-Test Split (80/20)
- Hyperparameter Tuning using GridSearchCV

## 🚀 Deployment
Built using Streamlit for real-time prediction.

## ▶️ How to Run
```bash

streamlit run app.py
